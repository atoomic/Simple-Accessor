# Simple-Accessor Development Notes

## Build & Test

```bash
perl Makefile.PL && make && make test
```

## Files to never commit

The following are build artifacts and must never be committed to git:

- `Makefile` / `Makefile.old` — generated by `perl Makefile.PL`
- `MYMETA.json` / `MYMETA.yml` — generated by `perl Makefile.PL`
- `blib/` — generated by `make`
- `pm_to_blib` — generated by `make`
- `.last-review-sha` — local review state file

These are already listed in `.gitignore`.
