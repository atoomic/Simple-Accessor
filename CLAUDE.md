## Project Overview

Simple::Accessor is a zero-dependency Perl module published on CPAN that provides lightweight object accessors. The entire implementation lives in a single file: `lib/Simple/Accessor.pm`.

## Build & Test

```bash
perl Makefile.PL && make && make test
```

Run a single test file:
```bash
perl Makefile.PL && make && prove -lv t/01-basic.t
```

## Distribution

Built with Dist::Zilla. The `Makefile.PL` in the repo is auto-generated by `dzil build` (via `[Run::AfterBuild]`) so CI can run without Dist::Zilla installed.

To build a release tarball:
```bash
dzil build
```

## Architecture

The module uses a package-level `$INFO` hash to track per-class metadata (declared attributes, composed roles, attribute origins). Key internal functions:

- `import()` — called on `use Simple::Accessor qw{...}`, registers attributes and installs `new`/`with`/accessors into the caller's namespace
- `_add_new()` — installs `new()` with support for `_before_build`, `build`/`initialize`, `_after_build`, and strict constructor
- `_add_accessors()` — creates getter/setter closures with hook dispatch (`_before_*`, `_validate_*`, `_after_*`) and lazy `_build_*` defaults
- `_add_with()` — installs `with()` for role composition, resolving transitive attribute origins

Multiple `use Simple::Accessor` calls in the same package accumulate attributes. Role attributes track their origin package so hooks resolve correctly through transitive composition.

## Test Helpers

Test helper modules live in `t/lib/` (e.g., `MyObject`, `Role::Age`, `TestRole`). Tests use `FindBin` + `lib` to load them.
